Tic Tac Toe

Reflections

  • Lessons learned: Building Tic Tac Toe reinforced state management, win-condition checks, and turn-based logic.
  • What I like: The clarity of the game logic and how quickly it demonstrates algorithmic thinking.
  • What I'd change: Add a simple AI, animation for moves, and a reset/scoreboard.
  • Future usefulness: This project helps with problem decomposition and testing small algorithms in bigger projects.
